# Friend Tracker App

## Description

Friend Tracker App is a React-based web application that helps users manage and maintain their personal relationships. Users can track interactions such as calls, texts, and video chats, view detailed friend information, and monitor communication goals through a timeline.

## Technologies Used

- React.js
- Tailwind CSS
- React Router
- Context API
- React Toastify

## Key Features

1. Interaction Timeline
   Track all calls, texts, and video interactions with friends in a structured timeline.

2. Friend Details Management
   View detailed information about each friend including status, tags, goals, and contact details.

3. Quick Check-In Actions
   Log interactions (Call, Text, Video) instantly with real-time timeline updates and notifications.
